# Progress Hub API 

**Progress Hub API** — це бекенд-додаток для управління особистими цілями, завданнями та відстеження прогресу. 

## 🛠 Технології

* **Мова:** Python 3.10+
* **Фреймворк:** FastAPI
* **База даних:** MySQL
* **ORM:** SQLAlchemy
* **Драйвер БД:** PyMySQL
* **Сервер:** Uvicorn

## 📂 Архітектура проєкту

Проєкт використовує `APIRouter` для розділення логіки на незалежні модулі:
* `/users` — управління користувачами.
* `/categories` — довідник категорій для цілей.
* `/goals` — створення та управління глобальними цілями.
* `/tasks` — дрібні завдання із системою пріоритетів та XP.
* `/progress` — логування відсотків виконання завдань (зв'язок 1:1).

Повну ER-діаграму бази даних можна переглянути в папці `DB/schema.md`.

2. Встановлення залежностей
